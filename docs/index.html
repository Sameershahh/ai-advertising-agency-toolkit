Task 2.3 — RAG System

Campaign Knowledge Bot

A CLI chatbot that answers questions strictly from uploaded agency documents. Refuses out-of-scope questions. Returns answer + source document + verbatim quote.

Architecture
01 / INGEST
PDF/TXT → 800-char chunks, 150-char overlap
02 / INDEX
Voyage-3 embeddings → NumPy cosine similarity
03 / GENERATE
Top-5 chunks → Claude (temp 0.1) → grounded answer

Anthropic API with 3-retry exponential back-off
def call_anthropic_with_retry(prompt, system="You are helpful.", model="claude-opus-4-5", max_tokens=1024, max_retries=3): client = anthropic.Anthropic() base_delay = 2.0 # seconds — doubles each attempt for attempt in range(1, max_retries + 1): try: message = client.messages.create( model=model, max_tokens=max_tokens, system=system, messages=[{"role": "user", "content": prompt}], ) return message.content[0].text except anthropic.RateLimitError: if attempt == max_retries: raise RuntimeError(f"Rate limit after {max_retries} retries.") delay = base_delay * (2 ** (attempt - 1)) # 2s → 4s → 8s time.sleep(delay) except anthropic.APIStatusError as exc: if exc.status_code >= 500 and attempt < max_retries: time.sleep(base_delay * (2 ** (attempt - 1))) else: raise # Non-retryable — surface immediately except anthropic.APIConnectionError: if attempt == max_retries: raise RuntimeError("Connection failed after all retries.") time.sleep(base_delay * (2 ** (attempt - 1)))
RateLimitError (429) APIStatusError (5xx) APIConnectionError Exponential back-off
